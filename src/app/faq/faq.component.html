FAQ's

  • 1. How can we find the current status of DCT ?

    Ans - Current state of the DCT can be checked in INFO tab. Current Owner in INFO tab tells about the current status of DCT.

  • 2. Can we change the the INPUT VALUE for Tranche1 DCT ?

    Ans - INPUT VALUE can be updated for modifiable DCT's but comment and document upload is mandatory to proceed further.

  • 3. Can we upload DCT data fro TRANCHE 2 and TRANCHE 3 DCT's, instead on manually entering ?

    Ans - Yes, upload feature is available in Tranche 2 and Tranche 3 DCT's. You can download the template and fill it up for your respective DCT's in excel and upload it back into the system.

  • 4. Can Approver change the value of any DCT ?

    Ans - No, Approver doesn't have the rights to change the value, he can only Approve or Reject it back to Maker with necessary comments.
